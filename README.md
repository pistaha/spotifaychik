# spotifaychik

`spotifaychik` — более ранняя итерация музыкального сервиса. По структуре проект похож на `musicserverlab2-course`, но здесь меньше инфраструктуры и меньше готового деплойного контура.

## Возможности
- многоуровневая структура решения;
- API, application/domain/infrastructure слои;
- тестовый проект;
- развитие идеи музыкального сервиса в учебном формате.

## Стек
- ASP.NET Core
- C#
- .NET

## Запуск локально
```bash
dotnet run --project MusicService.API/MusicService.API.csproj
```

Если проект использует БД и конфигурацию окружения, их нужно настроить отдельно.
